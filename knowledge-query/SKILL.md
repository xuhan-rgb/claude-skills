# 知识库查询 Skill

## 知识库位置
`~/.claude/knowledge/`

## 查询流程

1. **理解查询意图**：用户想查什么（公式？易错点？概念？）

2. **搜索知识库**：
   ```
   使用 Grep 工具搜索 ~/.claude/knowledge/ 目录
   - 搜索关键词、标签（tags:）、或分类（category:）
   - 使用 -i 忽略大小写
   - 使用 -C 3 显示上下文
   ```

3. **读取相关文件**：找到匹配后，用 Read 工具读取完整内容

4. **整理输出**：
   - 直接展示相关的公式/知识点
   - 高亮易错点（如果有）
   - 提供示例代码（如果有）

## 搜索策略

**按关键词**：
```
Grep pattern="四元数" path="~/.claude/knowledge/"
```

**按标签**：
```
Grep pattern="tags:.*quaternion" path="~/.claude/knowledge/"
```

**按易错点**：
```
Grep pattern="易错点" path="~/.claude/knowledge/" -A 10
```

## 输出格式

```markdown
## 📚 知识库查询结果

**来源**: `文件路径`

### 相关内容
[知识内容]

### 易错点提醒
- ⚠️ 易错点1
- ⚠️ 易错点2
```

## 注意事项

- 如果找不到精确匹配，尝试相关词搜索
- 展示知识时保留原始格式（公式、代码块）
- 如果用户问的内容知识库没有，建议添加
